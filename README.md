## 구현 내용

> ＋ , - 버튼을 이용하여 count가 올라가고 내려간다. (0 이하로는 내려가지 않는다.)

> 휴지통 버튼을 이용하여 목록에서 삭제한다.

> 입력창에서 목록의 이름을 입력하고 제출(Enter or AddBtn)하면 리스트에 올라간다.
